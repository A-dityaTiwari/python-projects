## 📖 Project Overview

**Mad Libs** is a classic word game where players provide words without knowing the context, which are then inserted into a pre-written story to create amusing narratives. This beginner-friendly Python implementation asks users for two adjectives, a noun, and a verb to generate a funny zoo adventure story!

### 🎯 What this project teaches:
- 🎨 **Basic user input** with the `input()` function
- 📝 **String formatting** using f-strings
- 💻 **Sequential programming** with simple Python statements
- 🎭 **Creative coding** that produces entertaining results!

## 🛠️ How to Run

### Prerequisites
- 🐍 **Python 3.x** installed on your system
- 💻 **Terminal/Command Prompt** access
- 🎯 **Ready to laugh!**

### 🎮 Quick Start Guide

1. **Clone the repository:**
   ```bash
   git clone https://github.com/A-dityaTiwari/python-projects.git
   cd python-projects/Mad-Libs
   ```

2. **Run the game:**
   ```bash
   python mad-libs.py
   ```

3. **Follow the prompts:**
   - Enter your first adjective
   - Enter your second adjective  
   - Enter a noun
   - Enter a verb
   - Enjoy your custom zoo story!

---

## 📋 Example Usage

```
$ python mad-libs.py

Enter the adjective: silly
Enter the adjective: purple  
Enter the noun: elephant
Enter the verb: dancing

I went to a silly zoo
I saw a purple elephant
It was dancing
```

### 🎪 Story Template

The game uses this fixed story format:
```
I went to a [adjective1] zoo
I saw a [adjective2] [noun]
It was [verb]ing
```

### 🎭 Sample Outputs:

**Input:** funny, green, monkey, jumping  
**Output:**
```
I went to a funny zoo
I saw a green monkey
It was jumping
```

**Input:** magical, sparkly, unicorn, flying  
**Output:**
```
I went to a magical zoo
I saw a sparkly unicorn  
It was flying
```
